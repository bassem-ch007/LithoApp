Demandes d'analyses

Les demandes d'analyses sont rattachées aux épisodes cliniques.

Accéder aux demandes d'analyses

Pour créer ou consulter les demandes d'analyses d'un patient, ouvrez le patient, sélectionnez un épisode clinique, puis utilisez l'action Demandes d'analyses sur la page de l'épisode.
